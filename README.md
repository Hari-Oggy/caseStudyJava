# TCP AND UDP USING IPV6

>##### description:

this is a simple **Tcp** and **Udp** project that using ipv6 

>the tcp server run on 9000 port number, and udp run 9001 

TO start the server you just run the command

> if you on windows
    go the directory that contains ***gradlew.bat*** and run the command :---

```bash
./gradlew.bat run
 ```
    
    
> if you on linux(ubuntu,kali,etc.....)
    go the directory that contains ***gradlew.bat*** and run the command :--- 
```powershell
./gradle.bat run
```
---

    then run the client side logic
